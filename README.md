# 🌐 Web Scraped Data Cleaning

## 📌 Overview
This project focuses on **cleaning and structuring messy data extracted from the web**.  
Web scraping often results in **incomplete, inconsistent, and noisy data** that must be cleaned before use.

## 🚀 Features
✅ Remove **duplicate or irrelevant scraped data**  
✅ Fix **broken or missing data fields**  
✅ Standardize **date formats and text formatting**  
✅ Extract **meaningful information from HTML & JSON files**  
✅ Export **cleaned data in CSV format**  

## 📂 Files in This Project
| File Name                     | Description                                      |
|--------------------------------|--------------------------------------------------|
| `web_scraped_data_cleaning.py` | Python script to clean web-scraped data.        |
| `raw_web_data.csv`             | Raw dataset collected from web scraping.        |
| `cleaned_web_data.csv`         | Output file with cleaned data.                  |

## ⚙️ How to Run
1. **Download the script and dataset** (`web_scraped_data_cleaning.py` & `raw_web_data.csv`).
2. Open a **Terminal or Command Prompt** in the project folder.
3. Run the script:
   ```bash
   python web_scraped_data_cleaning.py
